# Melodi v2.2.1

## Değişiklikler

### Yeni Özellikler
- **İndirme Desteği** — Şarkıları YouTube üzerinden indirme (FLAC için LosslessResolver + YouTube yedek)
- **İndirme Butonu** — Şimdi oynatılan ekranında ⬇️ butonu, indirme durumunu gösterir
- **Teşekkürler Bölümü** — Ayarlar → Teşekkürler (açık kaynak projeler listesi)
- **Otomatik Senkronizasyon** — Spotify/YT Music hesap bağlayınca playlistler otomatik çekilir

### Düzeltmeler
- **Senkronizasyon artık çalışıyor** — triggerManualSync() gerçek playlist/şarkı çekiyor
- **Material çökmesi düzeltildi** — mixes_screen'deki InkWell null hatası
- **Şarkı sözleri tek satır gösterimi** — karaoke stili, animasyonlu geçişler
- **İndirilen dosyalar** otomatik kütüphaneye ekleniyor
- **Playlist senkronizasyonu** tekrar senkronizasyonda güncelleniyor

### iOS İçin
```bash
cd /tmp/melodi
flutter build ios --release
```

APK: `release/Melodi-v2.2.1.apk` (63MB)
